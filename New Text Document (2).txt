- Nothing will be written in body and head tag in index.html for React app.
- App.js is main file of React app.
- Change language mode of App.js from javascript to javascript react (jsx).
-         jsx -------->(Nodejs)-------> HTML + CSS + JS
- React Router DOM (library)       https://www.npmjs.com/package/react-router-dom
    (run --->  npm i react-router-dom   in terminal of main folder of React app (like frontend and project1 in my case).
- Routing :-  Navigation from one component to other, helps in creating different addresses for different components.
- Double curly bracket (  {{}}  ) is used to insert js syntax in jsx.
- First letter of file in components folder MUST be capital.
